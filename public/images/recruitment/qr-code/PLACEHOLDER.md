# QR Code Placeholder

## Required QR Code

**Filename:** `zalo-hr-qr.png`

**Purpose:** Zalo contact QR code for HR department

**Specifications:**
- Size: 300x300px or 400x400px
- Format: PNG with transparent background
- Content: Link to INNO HR Zalo account

**Usage:** Displayed in job detail page contact section

---

## How to Add

1. Generate QR code for your Zalo HR account
2. Save as `zalo-hr-qr.png` in this folder
3. The code will automatically display it
